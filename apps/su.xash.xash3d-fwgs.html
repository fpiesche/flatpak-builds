Xash3D-FWGS is a fork of the Half-Life compatible Xash3D engine with many new features:

  • High portability to multiple platforms
  • Advanced multiplayer
  • Touchscreen and gamepad support
  • And much more

This is a Flatpak distribution of Ryan C. Gordon's Linux port of the game, for ease of use as the game can be difficult to run on modern systems.

PLEASE NOTE: This Flatpak does not include the Half-Life game data. You will need to supply it, either by installing it in Steam or copying the valve directory from an installation fo the game to $HOME/.var/app/su.xash.xash3d-fwgs/data/.
