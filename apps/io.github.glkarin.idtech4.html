Modernised source ports of the flagship idtech4 games: Doom 3, Prey (2006) and Quake 4.

PLEASE NOTE: This Flatpak does not include any game data or CD keys for these games. You will need to supply both for any of the games you wish to play. At least the *.pk4 files from the relevant game data must be copied to an appropriate sub-directory in $HOME/.var/app/io.github.glkarin.idtech4/[doom3|prey|quake4]/.

Launching any of the games included will inform you with a message if any of the required files are missing and where you need to copy them.
