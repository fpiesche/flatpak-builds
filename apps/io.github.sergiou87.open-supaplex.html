Open Supaplex is a reverse engineered multi-platform port of the early-90s puzzle game Supaplex.

Since then the source code has been lost. Some enthusiasts have disassembled the game, fixed many bugs, and added many features.

OpenSupaplex is a 1:1 reimplementation of the original game in C and SDL, with just a few changes to make it easy to use with game controllers, and also easy to port to any platform.
